I miei Progetti

Budget Manager

Un'applicazione web progettata per aiutare gli utenti a gestire le proprie finanze personali, monitorando le spese, impostando obiettivi di risparmio e visualizzando i modelli di spesa.

Link GitHub:

Vai alla Repository

Caratteristiche:

  • Registrazione e Autenticazione Utente
  • Tracciamento delle Spese (aggiungi, visualizza, elimina)
  • Obiettivi di Risparmio
  • Visualizzazione delle Spese con grafici interattivi
  • Gestione Profilo (salario, obiettivo di risparmio)
  • Cronologia Spese
  • Reset Spese

File Principali:

  • app.py: Logica principale dell'applicazione Flask.
  • helpers.py: Funzioni di utilità.
  • templates/: Directory dei template HTML.
  • static/: Directory dei file statici.
  • README.md: Panoramica del progetto.
